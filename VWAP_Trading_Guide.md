# دليل التداول — VWAP Bounce على NQ (5M)

---

## ① إعداد الشارت

```
┌─────────────────────────────────────────────────────┐
│  NQ1! / MNQ1!   │   5 Minutes   │   Regular Session │
│                                                      │
│  ████ Upper Band (VWAP + 1.5 ATR) ═══════════════  │  ← أحمر
│                                                      │
│  ─ ─ ─ ─ ─ ─ ─ VWAP ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  │  ← أصفر
│                                                      │
│  ████ Lower Band (VWAP - 1.5 ATR) ═══════════════  │  ← أخضر
│                                                      │
│  ░░░░░░░░░░░░░░░░░░░░ EMA 50 ░░░░░░░░░░░░░░░░░░░░  │  ← برتقالي
└─────────────────────────────────────────────────────┘
```

**الأدوات على الشارت:**
- VWAP (أصفر) — يُعاد حسابه كل يوم
- Upper Band (أحمر) = منطقة البيع
- Lower Band (أخضر) = منطقة الشراء
- EMA 50 (برتقالي) — فلتر الاتجاه

## ② إشارة LONG (شراء)

```
السعر ينزل ويلمس النطاق السفلي:

  │                                    │
  │    ┌──┐                            │
  │    │  │  شموع هابطة               │
  │    │  └──┐                         │
  │    │     └──┐                      │
  │════════════════ VWAP ══════════════│
  │             │  └──┐                │
  │             │     └┐               │
▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ Lower Band ▓▓│  ← النطاق الأخضر
  │                  │*│               │
  │                  │ │ ← شمعة الإشارة│
  │                  │ │   الذيل يخترق │
  │                  │ │   النطاق لكن  │
  │                  └─┘   الإغلاق فوقه│
  │                   ▲                │
  │               ▲ LONG ▲             │
  │              /                     │
  │    TP ══════╗  +2R               │  ← هدف الربح
  │             ║                     │
  │    دخول ════╬═══════════════════  │  ← سعر الإغلاق
  │             ║                     │
  │    SL ══════╝  -1R               │  ← تحت قاع الشمعة
```

### شروط إشارة LONG:
```
✅ Low ≤ Lower Band        (الذيل لمس النطاق)
✅ Close > Lower Band      (الإغلاق فوق النطاق)
✅ Close > Open            (شمعة خضراء صاعدة)
✅ Lower Wick ≥ ATR × 15%  (ذيل سفلي واضح)
✅ RSI ≤ 42                (مبالغة في البيع)
✅ وقت الجلسة 07:00-20:00 UTC
```

## ③ إشارة SHORT (بيع)

```
السعر يصعد ويلمس النطاق العلوي:

▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ Upper Band ▓▓│  ← النطاق الأحمر
  │                  │ │               │
  │                  │*│ ← شمعة الإشارة│
  │                  │ │   الذيل يخترق │
  │                  │ │   النطاق لكن  │
  │                  └─┘   الإغلاق تحته│
  │             ┌──┐                   │
  │           ┌─┘  │  شموع صاعدة      │
  │         ┌─┘    │                  │
════════════════════════ VWAP ═════════│
  │                                   │
  │                ▼ SHORT ▼           │
  │                                   │
  │    SL ══════╗  +1R  (فوق القمة)  │  ← وقف الخسارة
  │             ║                     │
  │    دخول ════╬═══════════════════  │  ← سعر الإغلاق
  │             ║                     │
  │    TP ══════╝  -2R  (هدف الربح)  │  ← هدف الربح
```

### شروط إشارة SHORT:
```
✅ High ≥ Upper Band       (الذيل لمس النطاق)
✅ Close < Upper Band      (الإغلاق تحت النطاق)
✅ Close < Open            (شمعة حمراء هابطة)
✅ Upper Wick ≥ ATR × 15%  (ذيل علوي واضح)
✅ RSI ≥ 58                (مبالغة في الشراء)
✅ وقت الجلسة 07:00-20:00 UTC
```

## ④ إدارة الصفقة

```
  مثال: LONG على MNQ

  سعر الإشارة:  21,000
  ATR:            25 نقطة
  
  SL  = قاع الشمعة - (ATR × 0.1)
      = 20,975 - 2.5
      = 20,972.5  ← خسارة محدودة

  Risk = 21,000 - 20,972.5 = 27.5 نقطة

  TP1 = دخول + (Risk × 2)
      = 21,000 + 55
      = 21,055  ← هدف الربح

  ┌─────────────────────────────────┐
  │  نقاط المخاطرة:  27.5          │
  │  نقاط الربح:     55.0          │
  │  نسبة R:R =      1 : 2         │
  │  معدل الفوز 55% → ربح صافي ✅  │
  └─────────────────────────────────┘
```

## ⑤ فلتر الاتجاه (EMA 50)

```
  الوضع المثالي للصفقات:

  LONG فقط عندما:                SHORT فقط عندما:
  ┌──────────────────┐           ┌──────────────────┐
  │  السعر > EMA 50  │           │  السعر < EMA 50  │
  │  (اتجاه صاعد)   │           │  (اتجاه هابط)   │
  │                  │           │                  │
  │  ░░░░ EMA 50     │           │  السعر هنا       │
  │        ↑         │           │        ↓         │
  │  ✅ LONG جيد    │           │  ░░░░ EMA 50     │
  └──────────────────┘           │  ✅ SHORT جيد   │
                                  └──────────────────┘

  ⚠️ المؤشر يعطي إشارة بدون هذا الفلتر
     لكن أضفه يدوياً لتصفية الإشارات الضعيفة
```

## ⑦ خطوات التنفيذ خطوة بخطوة

```
الخطوة 1: انتظر الشمعة 5M
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  → لا تتداول قبل إغلاق الشمعة
  → الإشارة تظهر عند إغلاق الشمعة ← مهم!

الخطوة 2: تحقق من الإشارة
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ✅ يظهر سهم ▲ أو ▼ على الشارت؟
  ✅ RSI في المنطقة الصحيحة؟
  ✅ الذيل واضح وطويل؟
  ✅ في وقت الجلسة النشطة؟

الخطوة 3: الدخول
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  → دخول عند فتح الشمعة التالية
  → Market Order أو Limit قريب من الإغلاق

الخطوة 4: وضع SL و TP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  → SL: تحت قاع شمعة LONG / فوق قمة SHORT
  → TP: ضعف المسافة للـ SL (R:R 1:2)

الخطوة 5: إدارة الصفقة
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  → لا تحرك SL قبل وصول السعر لنصف الهدف
  → عند نصف الطريق: حرك SL لنقطة الدخول (BE)
  → دع الصفقة تعمل حتى TP أو SL
```

## ⑧ ما يُفسد الإشارة (تجنبها)

```
  ❌ إشارة ضعيفة — تجاهلها:

  1. شمعة كبيرة جداً تغلق بعيداً عن النطاق
     السعر ──────────────────── النطاق
                              ↑
                          إغلاق بعيد جداً

  2. RSI في المنتصف (43-57) — إشارة غامضة

  3. قبل بيانات اقتصادية مهمة (NFP, CPI, FOMC)
     → أغلق كل الصفقات 15 دقيقة قبل النشر

  4. في آخر ساعة من الجلسة (بعد 19:00 UTC)
     → السيولة تقل والإشارات أقل دقة

  5. إذا كان السعر بين VWAP والنطاق (لا يلمس النطاق)
     → انتظر حتى يصل للنطاق الحقيقي
```
